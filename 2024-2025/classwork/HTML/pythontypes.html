python types

string

A string can be any text. Example: "funny"

integer

A integer can be any whole number. Example: 6

list

A liat stores more than one item. Example: words= ["crazy","funny","fun"] lists use an index starting at 0 to reference each item.

float

A float is a decimal number.Example: 1.5

dictionary

mutable data structures that allow you to store key-value pairs

tuple

a type of data structure that is very similar to lists.

set

a data type in python used to store several items in a single variable.

boolean

a Boolean variable can have only two values - True or False
